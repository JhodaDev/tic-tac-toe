# Tic Tac Toe

![bpxighj3oorldbr7wzaq](https://user-images.githubusercontent.com/43737400/201431297-9001dcb0-5d2b-4989-92c4-8f148f2af1b1.jpg)


Tic tac toe is a game of two players, X and O, taking turns placing their respective symbols on a 3x3 grid. The player who manages to place three of their symbols in a row, column or diagonal wins the game.

## how to install the tic tac toe game 

1. clone the repository - git clone
2. install the dependencies - npm install 
3. run the game - npm run dev

or you can find the game at the following link: https://tic-tac-toe-gamma-ten.vercel.app
